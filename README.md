smtp
====

smtp project for client server class

1. Build examples: 

    make example-client
    make example-server

2. Run examples:

    ./example-client <addr>
    ./example-server <port>

3. Build client/server:

    make client
    make server

4. Run client/server:

    ./smtp-client <server-addr>
    ./smtp-server <port>

3. Clean the build:

    make clean
